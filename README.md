# webscraper-for-icc-odi-rankings
a simple web scraper to scrap icc odi rankings.

Cricket Team Rankings Scraper
This Python script scrapes men's cricket team rankings from the ICC (International Cricket Council) website for One Day Internationals (ODIs) and stores the data in a CSV file.


Prerequisites
Before you begin, ensure you have met the following requirements:

Python 3.x installed on your system.
Required Python packages:
requests
BeautifulSoup (from bs4)
pandas

Customization
You can customize the script by modifying the following variables:

url: The URL of the ICC ODI team rankings page. You can change this to fetch rankings for other formats or categories.
head1: Customize the column headers in the CSV file as needed.
Adjust any other parameters in the script to fit your specific requirements.
